# FMPInvest

Projeto acadêmico da disciplina UC HUB: Projeto Prático.

## 📌 Sobre o projeto

O FMPInvest é uma plataforma web voltada para educação financeira e acompanhamento de investimentos.

## 🎯 Objetivo

Ajudar usuários a organizar finanças e acompanhar investimentos.

## 👥 Público-alvo

- Iniciantes em investimentos
- Pessoas que querem organizar finanças

## 🚧 Status

Sprint 1 em andamento

## ⚙️ Funcionalidades Iniciais

- Cálculo da taxa de investimento mensal
- Cálculo do score da carteira com base em diversificação e concentração
- Simulador de crescimento com juros compostos
- Estrutura inicial de backend para regras de negócio
- Planejamento de testes unitários para validação das funcionalidades

## Workflow de Versionamento

O grupo adotou o GitHub Flow para organização do projeto.

- `main`: branch principal do projeto
- cada integrante deve criar uma branch para suas alterações
- as alterações são feitas em branches separadas
- o merge na `main` ocorre via Pull Request
- outro integrante deve revisar antes do merge

### Padrão de nomes de branches

- `docs/nome-da-tarefa`
- `feat/nome-da-tarefa`
- `fix/nome-da-tarefa`
- `chore/nome-da-tarefa`

## Padrão de Commits

O grupo adotou o padrão Conventional Commits.

### Prefixos utilizados

- `docs:` documentação
- `feat:` novas funcionalidades
- `fix:` correções
- `chore:` organização
- `test:` testes

### Exemplos

- `docs: atualiza README com workflow`
- `chore: adiciona gitignore`
- `feat: inicia frontend`
- `test: adiciona planejamento de testes`

### Fluxo de trabalho

1. Criar uma nova branch a partir da `main`
2. Realizar alterações no projeto
3. Fazer commit seguindo o padrão definido
4. Enviar a branch para o GitHub
5. Abrir um Pull Request (PR)
6. Outro integrante revisa e comenta
7. Após aprovação, realizar o merge na `main`
